# MegaDex - Catálogo de Mega Evoluções

Projeto em React criado para a disciplina de Desenvolvimento Web.

## Como rodar

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

3. Abra o link exibido no terminal (geralmente `http://localhost:5173`).

O projeto utiliza React, React Router e CSS Modules.
